# Metamorphosis-gym
A repository for our gym project website that utilizes React, MySQL, Express, Tailwind CSS, and the PayPal API.

## Tech Stack

- Frontend: React.js
- Backend: Node.js/Express.js
- Database: MySQL
- Styling: Tailwind CSS
- Payments: PayPal API


## Dependencies
- ### Client
- react-icons
- react-router-dom
- jwt-decode
- react-modal
- @paypal/react-paypal-js
- ### Server
- nodemon
- express
- cors
- mysql2
- express-session
- jsonwebtokens
